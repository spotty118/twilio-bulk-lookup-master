# Product Overview

Bulk Lookup for Twilio is an enterprise-grade contact enrichment platform powered by Twilio Lookup API and 14+ data providers.

## Core Purpose
- Bulk phone number validation and enrichment via Twilio Lookup v2 API
- Business intelligence gathering from multiple providers (Clearbit, NumVerify, Hunter.io, etc.)
- Contact management with duplicate detection and quality scoring
- AI-powered natural language search using OpenAI, Anthropic Claude, or Google Gemini

## Key Features
- CSV import/export for bulk contact processing
- Background job processing via Sidekiq with intelligent retry logic
- Business discovery by zipcode using Google Places and Yelp
- CRM sync with Salesforce, HubSpot, and Pipedrive
- SMS and voice outreach via Twilio
- Real-time webhooks for status updates
- Trust Hub business verification
- Verizon 5G/LTE coverage checking

## Target Users
- Sales and marketing teams for lead enrichment
- Fraud prevention teams for risk scoring
- Customer intelligence operations
- Market research and analytics

## Important Notes
- NOT officially supported or maintained by Twilio
- Requires Twilio account with API credentials
- Multiple optional third-party API integrations available
